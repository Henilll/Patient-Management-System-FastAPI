About PatientMS

PatientMS is a modern Patient Management System built using FastAPI for the backend and HTML, Tailwind CSS, and JavaScript for the frontend.

This application allows healthcare providers or administrators to create, view, update, sort, and delete patient records efficiently through a clean and responsive interface.
